  <behavior>
    - launchCampaign(terms?: FundTermsConfig): creates fresh FundraisingCampaign with id=crypto.randomUUID(), uses generateLPProspects(), sets closeStatus="pre_marketing", committedAmount=0, calledAmount=0, terms=terms??DEFAULT_FUND_TERMS, targetAmount=fund.fundSize. Only works when gamePhase==="playing" and activeCampaign===null
    - pitchLP(prospectId: string): finds prospect in activeCampaign.prospects, calls calculatePitchOutcome(), updates prospect status, recalculates committedAmount via calculateTotalCommitted(), returns {success, reason?, newStatus?}. Fires toast via news. Only works when activeCampaign exists
    - advanceFundClose(): checks if committedAmount >= getFirstCloseThreshold (advances to first_close) or >= getFinalCloseThreshold (advances to final_close). Returns {newCloseStatus, committed}. Only advances forward, never backward
    - configureFundTerms(terms: Partial<FundTermsConfig>): merges into activeCampaign.terms AND updates fund.managementFeeRate, fund.carryRate, fund.hurdleRate, fund.gpCommit atomically (Pitfall 3 from research). Returns {success, reason?}. Blocked if closeStatus==="final_close"
    - completeFundClose(): terminal action. Pushes snapshot to history first. Increments fundNumber, skillLevel, rebirthCount. Resets fund economics (deployed, cashAvailable, totalFeesCharged, carryAccrued, totalDistributions, gpEarnings). Clears portfolio, dealPipeline, activeCampaign. Sets gamePhase="setup". Clears history (not undoable). Preserves playerProfile
  </behavior>
